# Flappy Bird Project

Dự án này được tạo để bạn tự code game Flappy Bird.

## Cách chạy

- Cài đặt pygame nếu chưa có:
  - pip install pygame
- Chạy file:
  - python main.py

## Ghi chú

- File logic và giao diện đang để trống dưới dạng placeholder.
- Bạn có thể bắt đầu bằng cách chỉnh sửa file main.py.
